# 修改目标体重为空测试完全空数据

## 一、修改目标

将 `generateMockWeightGoal()` 函数中的 `targetWeight` 改为 0，模拟新用户完全空数据的场景。

## 二、修改步骤

### 2.1 修改 generateMockWeightGoal 函数
**文件**：`constants/MockWeightData.ts`

**修改点**：
将 `targetWeight: 68` 改为 `targetWeight: 0`

```typescript
export function generateMockWeightGoal(): WeightGoal {
  const startDate = dayjs().subtract(30, 'day').format('YYYY-MM-DD');
  return {
    targetWeight: 0,  // 改为 0，表示未设置
    height: 175,
    startWeight: 74.0,
    startDate,
  };
}
```

## 三、预期效果

修改后，空数据状态将显示：

1. **WeightStatusCard**
   - 当前体重：`——`
   - 无变化值
   - 无 BMI 卡片
   - 目标卡片显示："🎯 目标 未设置"
   - 副标题："点击设置目标"
   - **无进度条**（因为 targetWeight 为 0）

2. **WeightEmptyState**
   - 正常显示引导内容

这样就完全模拟了新用户首次使用的场景。

## 四、恢复数据

测试完成后，使用 `git checkout constants/MockWeightData.ts` 恢复。
